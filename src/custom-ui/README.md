1. Directories should be named the urlPath to be replaced; ex: `serviceentries`
2. Inside the directory should be three other directories: `details` `form` `list`

Ex:

-- serviceentries
|-- details
|- details.html
|- details.js
|- details.css
|-- form
|- form.html
|- form.js
|- form.css
|-- list
|- list.html
|- list.js
|- list.css
